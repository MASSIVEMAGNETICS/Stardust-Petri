## How Growth Happens (The Petri Phases)

1. **Inoculation** — Your raw prompt is encoded into a bipolar hypervector (deterministic, seeded hashing + bundling of tokens).
2. **Seed Matching** — The prompt vector is compared (via cosine) against a library of archetypal seed crystals. Top matches are bundled in superposition.
3. **Incubation & Resonance Evolution** — A small population of variant hypervectors is created. Each generation:
   - Mutate (bit flips + small noise)
   - Crossover between high-fitness individuals
   - Evaluate resonance: symbolic execution of the implied structure + aesthetic scoring (docstring density, fractal self-similarity, graceful failure modes)
4. **Crystallization** — The highest-resonance vector "selects" and parameterizes the final template fragments. Placeholders are filled. A complete, runnable, beautifully documented project is born.
5. **Harvest** — The living system is materialized to disk (with git init optional). You receive the full lattice plus a resonance report explaining why this particular form emerged.

Everything is visible. You watch the population’s resonance field strengthen. You see which archetypes dominated. You understand the geometry of the decision.

## Quickstart (Fully Local)

```bash
git clone https://github.com/MASSIVEMAGNETICS/Stardust-Petri.git
cd Stardust-Petri
python -m venv .venv
source .venv/bin/activate

python stardust_petri.py
```

No keys. No internet after install. The dish is ready.
